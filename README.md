💀 Jumpscare Web

Website prank sederhana dengan efek jumpscare:

- Fake loading
- Fullscreen paksa
- Flash putih
- Gambar horror fullscreen
- Multi sound (3 layer)
- Efek shake

---

📁 Struktur File

index.html
scare.jpg
sound1.mp3
sound2.mp3
sound3.mp3

---

🚀 Cara Pakai

1. Download / clone project
2. Ganti file:
   - "scare.jpg" → gambar horror resolusi tinggi
   - "sound1.mp3" → suara utama (jumpscare)
   - "sound2.mp3" → suara tambahan
   - "sound3.mp3" → suara background looping
3. Buka "index.html" di browser
4. Klik layar → tunggu loading → selesai

---

⚙️ Fitur

- Fake loading dengan progress random
- Auto fullscreen setelah klik
- Flash putih sebelum jumpscare
- 3 audio dimainkan berurutan
- Efek getar layar (shake)

---

⚠️ Catatan

- Browser butuh interaksi (klik) untuk memutar audio
- Pastikan semua file ada di folder yang sama
- Gunakan file dengan kualitas tinggi agar efek maksimal
